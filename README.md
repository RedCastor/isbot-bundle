isbot bundle UMD
=================

[isbot][] Bundle UMD (JavaScript module that detects bots/crawlers/spiders via the user agent).

**[Demo][]**

## Installing
```
npm install isbot-bundle
yarn add isbot-bundle
```

## Usage

```js
window.isBot(navigator.userAgent);
```
